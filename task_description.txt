Есть очередь покупателей в супермаркете. Ваша задача написать функцию для подсчета общего времени, которое потребуется для обслуживания всех покупателей на кассах.

Функция принимает на вход: customers: последовательность положительных целых чисел - каждый элемент обозначает количество условных временных единиц, которые требуются для обслуживания данного покупателя. n: положительное целое число - число касс.

Примеры:

totalTime([5, 3, 4], 1)
# возвращает 12
# потому что n=1 и общее время - простая сумма времен для каждого покупателя

totalTime([10, 2, 3, 3], 2)
# возвращает 10
# потому что n=2, а второй, третий и четвертый покупатели в очереди закончат до того как обслужат первого 

totalTime([2,3,10], 2)
# возвращает 12

Оцениваться будет: время выполнения программы (самое важное!) ясность и лаконичность кода будет плюсом

Примечание: вы вправе использовать любые контейнеры и сторонние функции.
